# Stress Phases

Stress Phase shows where a correction currently lives under pressure.

It is not a score. It is a practice planning label.

## Phases

Block:

- Low pressure.
- Controlled setup.
- The player is learning the correction.

Guided:

- Coach-guided pressure.
- Defender or teammate gives a readable cue.
- The player must connect perception to action.

Scrimmage:

- Live or near-live pressure.
- Timing, spacing, and defensive decisions matter.
- The correction must survive play.

Game Ready:

- The behavior has shown up under game-like pressure.
- The coach can still reinforce it with clips and triggers.

## Archive Use

Phase retrieval should help coaches ask:

- Which corrections are still in Block?
- Which Guided clips need scrimmage testing?
- Which Scrimmage clips should repeat tomorrow?
- Which Game Ready clips should be reviewed before a game?
