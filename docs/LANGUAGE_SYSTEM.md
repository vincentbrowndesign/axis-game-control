# Language System

Axis language is reward-first and observational.

## Primary Language

Use:

- START MEMORY
- CREATE DIGITAL TWIN
- MEMORY STORED
- REPLAY READY
- WARMUP ADDED
- ARCHIVE ACTIVE
- DIGITAL TWIN
- MEMORY OWNER
- LOCAL PLAYER
- BASKETBALL READ
- READ BUILDING
- COMPARISON AVAILABLE

## Secondary Language

Use after reward:

- 2 more warmups to unlock comparison.
- Axis is still building this read.
- Reads improve after more warmups.
- Movement stored.
- Rhythm captured.

## Avoid

- PLAYER UNASSIGNED
- Signal unavailable
- Not enough signal
- Control lost
- Pressure spike
- Run detected
- Recovery window
- skill grade
- fatigue
- basketball IQ
- shot quality
- decision quality

## Tone

Axis should feel calm, premium, sparse, and basketball-native.

Do not use generic sports analytics language.
