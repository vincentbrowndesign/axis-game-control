# Surface Situations

Surface situations are repeated basketball geometries that show up across practices, scrimmages, and games.

Axis organizes clips around these situations so coaches can retrieve the same correction when it matters again.

## Core Situations

- High Pick-and-Roll.
- Weak-Side Tag.
- Corner Kick.
- Slot Drive.
- Nail Help.
- Baseline Drift.
- Closeout Attack.
- Paint Touch.
- Ball Screen Reject.
- Transition Advantage.

## Why Situations Matter

Drill names often hide the real basketball problem.

Situation tags keep the clip connected to:

- Spacing.
- Defender location.
- Help responsibility.
- Passing window.
- Driving angle.
- Finishing decision.
- Next rotation.

## Example Tags

High Pick-and-Roll:

- Rounded off screen.
- Came off too wide.
- Missed pocket pass.
- Did not reject high defender.

Weak-Side Tag:

- Late low-man help.
- Lost corner shooter.
- Did not recover on air time.

Slot Drive:

- Picked up dribble early.
- Missed corner kick.
- Drove into nail help.

Closeout Attack:

- Stood tall on catch.
- First step was wide.
- Missed one-more pass.
