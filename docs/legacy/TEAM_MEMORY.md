# Team Memory

Teams are connected memory ecosystems.

Team memory connects:

- roster
- shared practice memory
- player continuity
- coach retrieval
- drill assignment
- progression tracking
- practice planning

## Coach Use

Coaches should retrieve useful continuity:

- clips to repeat
- drill history
- shared rhythm
- practice-to-game transfer
- team accountability

Do not build a stat dashboard. Build a retrieval advantage.
