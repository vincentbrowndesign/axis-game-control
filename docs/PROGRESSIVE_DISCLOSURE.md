# Progressive Disclosure

The visible Axis world should stay simple.

## Law

Complexity underneath. Clarity on the surface.

## Surface Priority

Show:

- next practice
- useful memory
- retrieval result
- continuity link
- team connection

Hide:

- raw system state
- inference machinery
- upload mechanics
- excessive metadata
- dashboard-style summaries

## Rule

Do not prove intelligence with clutter.

Let memory retrieval make the system feel intelligent.
