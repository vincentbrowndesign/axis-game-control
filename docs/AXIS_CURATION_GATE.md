# Axis Curation Gate

Before adding anything to an Axis page or loop, answer these questions:

1. What is the singular essence of this loop?
2. What is the one action the user needs?
3. What is the one result the system must produce?
4. What dataset does this create or update?
5. What provider powers it, if any?
6. What must stay hidden from the surface but visible in Axis Space?
7. What is the acceptance test?
8. What are we explicitly not adding yet?

## Rules

If it does not help this loop complete, it cannot appear on the page.

If it matters to the system but not the user right now, it goes in Axis Space.

If it cannot be tested, it is not active.

If it is future capability, it is planned, not visible.
