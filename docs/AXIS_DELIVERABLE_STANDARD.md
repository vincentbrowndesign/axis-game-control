# Axis Deliverable Standard

A loop is not complete because it works.

A loop is complete when it creates a market-facing deliverable.

Do not call it an artifact.

Do not call it an export.

Do not expose JSON as the value.

## Required Definition

Each loop must define:

1. User
2. Market deliverable
3. Hidden dataset
4. Proof it is real
5. What stays off the surface

## Rule

If a player, parent, coach, trainer, or program would not want to keep it, share it, review it, or act on it, it is not the deliverable.
