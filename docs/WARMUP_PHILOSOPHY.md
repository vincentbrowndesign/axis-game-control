# Warmup Philosophy

Warmups are daily discipline disguised as memory refinement.

The user feels:

- I am getting loose.
- I am starting my session.
- I am building rhythm.
- My archive is growing.

The system quietly improves:

- segmentation quality
- baseline cleanliness
- comparison reliability
- future perception reads

## Warmup Is Not

- a file upload
- a test
- a grade
- a training-data request
- a coaching diagnosis

## Warmup Is

- basketball-first
- repeatable
- low-friction
- memory-building
- identity-anchored

The warmup chain is the core progression unit.
