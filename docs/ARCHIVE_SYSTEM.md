# Archive System

The archive is remembered effort.

It is not a clip dump.

## Archive Actions

- Keep Memory
- Discard Memory
- Mark Useful

## Archive Grouping

Memories may group by:

- player
- team
- warmup
- drill
- practice block
- repeated rhythm
- recurring reset
- transfer pattern

## Rule

The archive should become more useful as memories connect.

Do not optimize for more clips. Optimize for better retrieval.
