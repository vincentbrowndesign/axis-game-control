# Retrieval Engine

Retrieval answers:

`What matters right now?`

It does not answer:

`What score did the player get?`

## Retrieval Intents

- strongest memories
- rhythm stabilization
- repeated resets
- memories to revisit tomorrow
- continuity streaks
- transfer clips
- current issue

## Rules

- Rank by relevance, not performance.
- Use observable memory structure.
- Do not invent coaching claims.
- Do not expose raw machinery by default.
- Prefer recognition over explanation.
