# Practice Loop

Axis starts with practice.

## Loop

1. Practice.
2. Record.
3. Keep meaningful memory.
4. Retrieve patterns.
5. Repeat intentionally.
6. Connect teammates.
7. Apply during scrimmage.
8. Observe transfer.
9. Reinforce next practice.

## Rule

Practice should feel like remembered effort becoming structure.

Do not frame practice as data collection.
