# Continuity System

Continuity is the moat.

Most basketball tools track isolated events. Axis tracks what keeps returning.

## Continuity Objects

- Player continuity
- Structured memory lanes
- Practice lineage
- Memory connections
- Team continuity
- Practice-to-game transfer

## Continuity Signals

Axis may connect memories through observable structure:

- same player
- same warmup
- repeated rhythm
- recurring reset
- recurring cadence
- repeated movement structure
- same practice block
- practice-to-game transfer

Do not turn continuity into skill judgment.
