# Axis Reference Lock

Axis should inherit visual DNA from:

- Apple Fitness
- Oura
- PS5 UI
- Xbox cards
- premium wearable interfaces
- athletic operating systems
- soft rounded modules
- progression calendars
- save-state systems

Axis should avoid:

- SaaS admin panels
- startup landing pages
- cyberpunk terminals
- analytics dashboards
- esports/casino leaderboard energy
- dead black cinematic voids
