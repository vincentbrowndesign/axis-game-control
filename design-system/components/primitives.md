# Axis UI Primitives

Source components live in `components/axis-design-system`.

Use these primitives for new UI before creating custom screen styling:

- `AxisSurface`
- `AxisHero`
- `AxisCard`
- `AxisSignal`
- `AxisMetric`
- `AxisCheckInState`
- `AxisProgressNode`
- `AxisCalendar`
- `AxisLeaderboardItem`

Rules:

- Compose screens from primitives first.
- Keep one dominant action per screen.
- Use tokens from `design-system/tokens/axis-tokens.css`.
- Do not introduce random inline colors, shadows, radii, or spacing.
- Do not turn primitives into dashboard widgets.
