# UI Context

Axis visual direction:

- Calm.
- Operational.
- Cinematic.
- Embedded.
- Structurally stable.
- Upload-first.
- Media-first.
- Intelligence underneath.

Current UI rule:

Do not redesign while stabilizing the upload -> session -> processing seam.

Use the existing Axis visual language:

- Black, charcoal, smoke gray, white.
- Subtle transparency.
- Restrained glow only where already established.
- Minimal typography.
- Structural layout, not dashboard stacks.

Avoid:

- Dashboard-heavy surfaces.
- Loud gradients.
- New visual experiments.
- Extra panels for clips, broadcast, overlays, or commentary.
